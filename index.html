Project Kaida

The server is currently locally ran on Aarotron9000's PC.
It is online whenever someone wants to come on and Aarotron9000 is available to boot the server up.

Core Concept

The main idea behind the creation of project Kaida was simply to form a forever vanilla survial world for friends.
That's it really. Just some friends playing survival Minecraft whenever they feel like it. ^^

Server Foundation

The server was intitially established by AaroTron9000, ZakZNinja & X_Kazuma_X.

Main Bases

> Build name by Player name.

Description.

> Pyramid by AaroTron9000.

A massive XXbyXX block pyramid surrounded by obelisks, with some texturing on the outside along with some quartz near its top.
There are many builds on the inside including AaroTron9000's storage, mass smelter, LOZ 3DS build by X_Kazuma_X, iron farm, etc.
A fantasy Egyptian/20th Century British village along with a trading dock and ship.

> Barad-Dur by ZakZNinja.

Saurons tower.

> COC TH2 by X_Kazuma_X.

A Clash of Clans themed base. Its a Town Hall 2 village with all buildings included.
The layout of the village was created by X_Kazuma_X in COC and used as a reference image throughout the design process.
AaroTron9000 built a Christmas house with an 80's style basement expansion for the base.

> HollowMoon by Archer.

A vast fantasy village composed mostly of sightly if at all edited tutorial builds with the odd custom desiged build.
It includes a dock, town hall, stable, villager trading hall, lighthouse, arena and much more.

> Blossom House by Tennozou.

A nice, quaint wooden house shrouded within a circular pink blossom forest.
